## Getting started 
Node 14 required

### modules installation
```shell
$ npm install
```
## .env 
```shell
After the creation of one database via MySQL SGBDD named DB_name, .env required with:
the parameters fixed during MySQL database creation + KEY_Token : needed to create the user token
PORT = ...
DB_host = ...
DB_user = ...
DB_user_pw = ...
DB_name = ...
KEY_token = ...
```
